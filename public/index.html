Welcome to my REST API

  • GET

    • ".../users" to get an array of users in JSON format.
    • ".../users?role=admin" or "...role=user" to get users corresponding to the role.
    • ".../users/user_id" to get one user. User_id must be a number bigger than 0.

  • POST

    • Send the POST request to ".../users" to create a new user.
    • Request body must be an object with the next fields: 
        {
    username: string,
    first_name: string,
    last_name: string,
    email: string,
    role: 'user' | 'admin',
    state: 'male' | 'female'
  }

  • PUT

    • Send the PUT request to ".../users/user_id" to update the user.
    • Request body must be an object with the next fields: 
        {
    username: string,
    first_name: string,
    last_name: string,
    email: string,
    role: 'user' | 'admin',
    state: 'male' | 'female'
  }

  • DELETE

    • Send the DELETE request to ".../users/user_id" to delete the user.
